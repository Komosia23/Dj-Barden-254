# DJ BARDEN 254 Official Website

![DJ BARDEN 254](images/header-bg.jpg)

## Description
This is the official website for **DJ BARDEN 254**, a professional DJ delivering electrifying music experiences for clubs, parties, and events.  
The website features a neon-light theme, interactive booking system, photo gallery, music mixes, upcoming events, and social media links.

---

## Features
- **Neon-themed design** with glowing effects for buttons, headings, and sections  
- **Responsive layout** for desktop and mobile devices  
- **Interactive booking form** with popup for direct event inquiries  
- **Gallery section** showcasing event photos  
- **Music mixes** embedded via SoundCloud  
- **Upcoming events** section to display DJ performances  
- **Social media links** for Instagram, Facebook, and Email
